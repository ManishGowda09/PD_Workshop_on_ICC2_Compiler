# PD_Workshop_on_ICC2_Compiler

Physical Design (PD) Workshop

> Hands-on learning repository covering the complete ASIC Physical Design flow using **Synopsys IC Compiler II (ICC2)**.

---

## 📖 About

This repository contains my notes, lab exercises, screenshots, and learning outcomes from an **ASIC Physical Design Workshop**.

The workshop focused on understanding the complete backend VLSI design flow using industry-standard EDA tools. Throughout the sessions, I learned Linux fundamentals, design planning, floorplanning, placement, routing, and physical verification.
